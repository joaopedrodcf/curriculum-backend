# curriculum-backend

This is the backend that will serve the curriculum project with information

## Getting Started

First just

```
git clone https://github.com/joaopedrodcf/blog.git
```

Then

```
npm install
```

Finally

```
npm start
```

Now its ready to use

### Prerequisites

You need to install Node.js
